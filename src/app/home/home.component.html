Home Page

A simple site to demonstrate Angular.

Site navigation: Please use the Side Nav.

Tracking ISS: Tracking the International Space Station's currnet location, courtesy WTIA REST API.

Google Routes: Displaying the route polylines on a google map, using google maps javascript API and directions API. User can select source and destination to display the road route between them.

Superbowl stats: A simple chart and a data table displaying the winners of the NFL superbowl from years 1967 to 2020. This page was build using chart.js javascript library and Angular material table. Dataset courtesy, Tim Bozsolik on kaggle.com
