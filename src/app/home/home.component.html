Home Page

A simple app to demonstrate Angular.

Site navigation: Please use the side-nav.

Tracking ISS: Tracking the International Space Station's current location, courtesy WTIA REST API and the open source interactive maps JavaScript library called Leaflet.

Google Routes: Displaying the route polylines on a google map, using google maps javascript API and directions API. User can select source and destination to display the road route between them. On this page load, please enable location services to your browser.

Superbowl stats: A simple chart and a data table displaying the winners of the NFL superbowl from years 1967 to 2020. This page was build using chart.js JavaScript library and Angular material table. Dataset courtesy, Tim Bozsolik on kaggle.com
